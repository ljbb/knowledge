# 知识库状态检查

## 触发条件
当用户询问知识库状态、知识库健康、知识库有什么问题时。

## 执行流程
1. 调用健康检查：
   ```bash
   python kb.py check
   ```
2. 分析检查报告：
   - A 级（基础）：必填字段、死链接、Markdown 语法
   - B 级（内容）：步骤可操作性、验证证据、回滚方案
   - C 级（全面）：新鲜度、交叉引用、孤立检测
3. 汇总问题并按优先级排列
4. 对每个问题给出修复建议
